## Application Overview

FuelTest is a Laravel-based web application for managing fuel quality testing records. Key features include:

- **User Management**: Authentication and role-based access
- **Fuel Testing**: Record creation, editing, and management
- **Vendor Management**: Supplier information tracking
- **Reporting**: Excel exports and PDF certificates
- **Analytics**: Statistics and insights dashboard
